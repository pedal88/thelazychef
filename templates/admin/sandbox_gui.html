Scaling Workarounds

If the window physically bleeds off your laptop screen, here are three ways to iterate without changing the core 1080x1920 CSS (which would break the headless generator):

1. CSS 'Scale' Injection (Recommended)

When the window opens, inject the `scale` CSS property directly into the URL by clicking below. This physically shrinks the rendered layout without altering the core templates.

2. Chrome Device Toolbar (The precise way)

Open the standard window, press Cmd + Option + I to open DevTools, then click the "Toggle Device Toolbar" icon (or Cmd + Shift + M). Enter "1080 x 1920" as a custom device. Chrome will auto-scale it to fit, and you can inspect elements perfectly.

3. Standard Browser Zoom

Open the standard window, hold Cmd and press - (minus) repeatedly to zoom the browser out to 50% or 33%.
